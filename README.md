# 🔒 OZON Hiding Address

Tampermonkey-скрипт для скрытия вашего адреса на страницах [ozon.ru](https://ozon.ru) — три независимых режима: размытие, удаление и подмена на случайный адрес.

## ⚙️ Возможности

- 🌫️ **Blur** — размывает текст адреса на странице
- 🗑️ **Del** — скрывает блок с адресом целиком (без следов в видимой части DOM)
- 🎭 **Spoof** — подменяет адрес на случайный российский (загружается через API, сохраняется между сессиями)
- 🔁 Кнопка обновления подменного адреса
- ♻️ Сброс всех режимов одной кнопкой
- 📌 Перемещаемое и сворачиваемое окно интерфейса
- 👁️ Кнопка-иконка в правом нижнем углу (можно скрыть — открывается через меню Tampermonkey)

## 📷 Интерфейс

Окно открывается нажатием кнопки 🔒 в правом нижнем углу экрана.

- Режимы **Spoof**, **Blur**, **Del** — независимые переключатели, можно включать любую комбинацию.
- При включённом Spoof показывается текущий подменный адрес и кнопка для получения нового.
- **🗑 Сбросить всё** — выключает все режимы и восстанавливает оригинальный адрес.

## 🧩 Установка

### 1. Установите расширение Tampermonkey

| Браузер | Официальная ссылка |
| --- | --- |
| **Chrome** | https://www.tampermonkey.net/index.php?browser=chrome |
| **Firefox** | https://www.tampermonkey.net/index.php?browser=firefox |
| **Microsoft Edge** | https://www.tampermonkey.net/index.php?browser=edge |
| **Opera** | https://www.tampermonkey.net/index.php?browser=opera |
| **Safari** | https://www.tampermonkey.net/index.php?browser=safari |
| **Yandex Browser** | Устанавливается как Chrome-расширение |

### 2. Установите скрипт

1. Откройте Tampermonkey
2. Создайте **новый скрипт**
3. Замените содержимое редактора на код из `ozon-hiding-address.user.js`
4. Сохраните (Ctrl+S)
5. Перейдите на OZON — кнопка 🔒 появится в правом нижнем углу

## 🚀 Использование

1. Перейдите на [ozon.ru](https://ozon.ru)
2. Нажмите кнопку 🔒 **Адрес** в правом нижнем углу
3. Включите нужные режимы:
   - **Spoof** — адрес будет подменён на случайный
   - **Blur** — адрес будет размыт
   - **Del** — блок с адресом будет скрыт
4. Для сброса нажмите **🗑 Сбросить всё**

## 🔒 Политика конфиденциальности

Скрипт обращается только к `api.randomdatatools.ru` для получения случайного адреса в режиме Spoof. Никакие личные данные не передаются.
