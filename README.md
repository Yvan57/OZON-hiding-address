# 🔒 OZON Hiding Address

Tampermonkey / Violentmonkey скрипт для скрытия адреса доставки на страницах [ozon.ru](https://ozon.ru) — четыре независимых режима: размытие, удаление, подмена и скрытие ПВЗ.

## ⚙️ Возможности

- 🌫️ **Blur** — размывает текст адреса
- 🗑️ **Del** — скрывает блок с адресом целиком
- 🎭 **Spoof** — подменяет адрес на случайный российский (загружается через API, сохраняется между сессиями)
- 📦 **ПВЗ** — скрывает адреса и номера пунктов выдачи в меню выбора ПВЗ
- 🔁 Кнопка обновления подменного адреса
- ♻️ Сброс всех режимов одной кнопкой
- 📌 Перемещаемое и сворачиваемое окно интерфейса
- 👁️ Кнопка-иконка в правом нижнем углу (можно скрыть — восстанавливается через меню Tampermonkey)

## 📷 Интерфейс

Окно открывается нажатием кнопки 🔒 в правом нижнем углу экрана.

- Режимы **Spoof**, **Blur**, **Del** — независимые переключатели для основного адреса, можно включать любую комбинацию.
- При включённом Spoof показывается текущий подменный адрес и кнопка для получения нового.
- Режим **ПВЗ** — отдельный переключатель для меню выбора пункта выдачи. Взаимодействует с основными режимами:
  - ПВЗ + Del — скрывает строки с адресом и номером ПВЗ
  - ПВЗ + Spoof — подменяет адрес ПВЗ на spoof-адрес, номера заменяет на случайные цифры
  - Только ПВЗ — размывает адреса, номера заменяет на случайные цифры
- **🗑 Сбросить всё** — выключает все режимы и восстанавливает оригинальный адрес.

## 🚀 Использование

1. Перейдите на [ozon.ru](https://ozon.ru)
2. Нажмите кнопку 🔒 **Адрес** в правом нижнем углу
3. Включите нужные режимы:
   - **Spoof** — адрес будет подменён на случайный
   - **Blur** — адрес будет размыт
   - **Del** — блок с адресом будет скрыт
   - **ПВЗ** — адреса и номера в меню выбора пункта выдачи будут скрыты
4. Для сброса нажмите **🗑 Сбросить всё**

## 🔒 Политика конфиденциальности

Скрипт обращается только к `api.randomdatatools.ru` для получения случайного адреса в режиме Spoof. Никакие личные данные не передаются и не сохраняются за пределами браузера.
